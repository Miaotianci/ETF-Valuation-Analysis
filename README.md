# ETF 趋势与估值分析看板

这是一个专业且直观的 ETF 投资决策辅助工具，支持 A股、港股、美股 ETF 的核心指标实时分析。通过深度整合多方金融数据源，帮助投资者快速识别标的的“安全边际”与“性价比”。

## 核心功能

- **全市场覆盖**：支持 A股、港股、美股主流 ETF 的搜索与深度分析。
- **多维估值体系**：
  - **PE/PB 百分位**：基于历史 5-10 年数据，判断当前估值所处的历史分位。
  - **最大回撤分析**：计算过去 10 年的历史回撤，识别风险释放程度。
  - **股息率与 ROE**：衡量标的的底层盈利能力与现金分红价值。
- **量化评分系统**：
  - **估值分 (0-12分)**：估值分位越低，得分越高。
  - **回撤分 (0-8分)**：回撤越大，逆向投资价值越高，得分越高。
  - **综合得分 (0-20分)**：直观判定“极度低估”、“低估”、“合理”或“高估”状态。
- **智能历史记录**：
  - 自动保存查询历史，并直接在侧边栏展示历史查询时的核心指标（PE、PB、股息率）。

## 技术架构

### 前端 (Frontend)
- **React 19 + Vite**：高性能的前端开发与构建环境。
- **Tailwind CSS**：打造极简暗黑风格，适配全设备响应式布局。
- **Recharts**：渲染平滑的历史净值趋势图。
- **Motion (Framer Motion)**：提供流畅的界面交互与加载动画。

### 后端 (Backend)
- **Node.js + Express**：轻量级高性能服务端。
- **多级数据回退机制 (A股优化)**：
    - **一级**：蛋卷基金/东方财富常用估值列表。
    - **二级**：蛋卷基金指数详情 API（深度覆盖细分行业指数）。
    - **三级**：东方财富实时财务接口（确保冷门品种的基础数据覆盖）。
- **Yahoo Finance API**：获取全球市场的历史 K 线数据与实时行情。
- **Node-Cache**：内存级缓存优化，显著提升二次查询响应速度。

## 快速开始

### 开发环境运行
1. 安装依赖：
   ```bash
   npm install
   ```
2. 启动开发服务器：
   ```bash
   npm run dev
   ```
   访问 `http://localhost:3000` 即可开始使用。

### 生产环境部署
1. 构建项目：
   ```bash
   npm run build
   ```
2. 启动服务：
   ```bash
   npm run start
   ```

## API 说明

### 获取 ETF 综合分析数据
- **路径**: `GET /api/etf/:market/:code`
- **参数**:
  - `market`: 市场代码 (`A`, `HK`, `US`)
  - `code`: ETF 代码 (如 `510300`, `SPY`, `2800`)
  - `interval` (query): 时间间隔 (`1d`, `1wk`, `1mo`)，默认为 `1mo`
